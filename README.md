# AIPASS

AI-Powered Adaptive Support Assistant System

## Overview

AIPASS is an AI-powered adaptive support assistant designed for remote laboratory learning environments.

The system provides:

- Real-time AI assistance
- Teacher-in-the-loop validation
- Adaptive learner profiling
- Learning analytics
- AI behaviour configuration
- WebSocket notifications

## Technology Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- Socket.io Client
- Recharts

### Backend
- NestJS
- TypeORM
- PostgreSQL
- JWT Authentication
- Socket.io
- Groq API (LLaMA 3.1)
